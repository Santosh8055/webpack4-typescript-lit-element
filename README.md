> A very simple and basic starter boilerplate template based typescript webpack 4 and lit-element

### 🏠 [Homepage](https://github.com/Santosh8055/webpack4-typescript-lit-element#readme)

## Install

```sh
yarn install
```

## Usage

```sh
yarn start:dev
```

## Run tests

```sh
npm run test
```
